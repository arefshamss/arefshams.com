این پروژه یک وبسایت رزومه شخصی مبتنی بر پورتفولیو است که با استفاده از ASP.NET Core و معماری ۵ لایه توسعه داده شده است. این وبسایت به‌صورت ماژولار و قابل نگهداری طراحی شده و سوابق فنی، مهارت‌ها، تحصیلات و پروژه‌های قبلی من را نمایش می‌دهد. این پروژه با تمرکز بر مقیاس‌پذیری و حرفه‌ای بودن، نمونه‌ای واقعی از به‌کارگیری معماری تمیز در یک ابزار برندینگ شخصی است.

فناوری‌های استفاده‌شده

  • #C

  • ASP.NET Core

  • Entity Framework Core

  • Microsoft SQL Server

  • معماری تمیز ۵ لایه

  • مپینگ دستی (بدون AutoMapper)

  • Google reCAPTCHA v2

  • Google reCAPTCHA v3

  • Bootstrap

  • jQuery

  • طراحی ماژولار برای توسعه و گسترش آسان امکانات

امکانات کلیدی

  • درباره من

    بخشی کامل برای معرفی پیشینه، اهداف شغلی و پروفایل شخصی

  • خدمات ارائه‌شده

    بخشی منعطف برای نمایش خدماتی که به‌عنوان برنامه‌نویس ارائه می‌دهم

  • رزومه، تحصیلات و دوره‌ها

    نمایش کامل اطلاعات تحصیلات رسمی و دوره‌های آموزشی

  • نمای کلی مهارت‌ها

    نمایش دسته‌بندی‌شده و گرافیکی مهارت‌های فنی

  • پروژه‌های انجام‌شده

    بخش پورتفولیو برای نمایش پروژه‌های واقعی پیاده‌سازی‌شده

  • فرم تماس

    امکان ارتباط کاربران از طریق فرم تماس

    افزایش امنیت فرم تماس با پیکربندی داینامیک CAPTCHA

  • مدیریت نقش‌ها و سطوح دسترسی

    پشتیبانی در سمت بک‌اند برای کنترل مدیریتی و محدودسازی دسترسی‌ها

درباره ریپازیتوری
